Doppl is a set of tools and libraries built on top of Google's J2objc. While J2objc is intentionally neutral with regards to Android, the Doppl ecosystem is designed to facilitate sharing logic and architecture between Android and iOS.

Status

Doppl has been running under "technical preview" for the past several months. We've been gathering feedback and making modifications to the build system.

Production Ready

"Doppl" had a slightly modified J2objc build, but the modifications were really minor. Doppl runs fully stock J2objc as of latest source and presumably whenever version 2.0.6 is released, the downloaded stock version. The build system is going through revisions, but J2objc itself, and the libraries we're building on, are used in production with very high profile applications from a number of companies.

The libraries are Android community standards, the Android architecture code is from AOSP directly. Doppl as an architectural sharing framework is production ready.
